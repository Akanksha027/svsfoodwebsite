# Bagoss font files

Home and locations pages use **Bagoss Standard** from [Displaay Type Foundry](https://displaay.net/typeface/bagoss).

## Included files (site uses these only)

```
public/fonts/bagoss/Bagoss Standard/
  BagossStandard-TRIAL-Regular.woff2   (400)
  BagossStandard-TRIAL-Medium.woff2    (500)
  BagossStandard-TRIAL-SemiBold.woff2  (600)
  BagossStandard-TRIAL-Bold.woff2      (700–900)
```

Other trial variants (Condensed, Extended, italics, OTF/TTF) were removed to keep the deploy size small.
